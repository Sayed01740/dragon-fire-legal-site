# Dragon Fire Legal Site

This repository contains the official legal documentation for the **Dragon Fire** decentralized application.

## 🚀 Live Site
This site is designed to be hosted via **GitHub Pages**.

## 📑 Contents
The site covers three critical areas:
- **License Agreement**: Usage terms and Solana Gaming Pass activation.
- **Copyright Policy**: IP protection for game assets and branding.
- **Privacy Policy**: Data collection (or lack thereof) and blockchain transparency.

## 🎨 Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Premium dark-mode design with glassmorphism.
- **Assets**: Custom AI-generated game banners and logos.
